### Jackson Library Install And Add

#### Library List

- jackson-annotations
- jackson-core
- jackson-databind

#### How to Install

- [com/fasterxml/jackson/core](https://repo1.maven.org/maven2/com/fasterxml/jackson/core/)

#### How to Add

- `commend + ;` > `Libraries` -> `+` -> `java` -> `Select the .jar file inside the libs directory`
